## 🎯 Зачем ещё один планировщик?

Потому что остальные:
- 💀 Просят **₽299/мес** за базовые функции
- 📱 Заставляют ставить **3-4 отдельных приложения** (задачи + бюджет + таймер + заметки)
- 🌐 **Не работают без интернета**
- 📡 **Отправляют ваши данные** на серверы

**Todo Budget** — это один лёгкий инструмент (8.5 МБ), который делает всё перечисленное бесплатно, оффлайн, с хранением данных только на вашем устройстве.

## ✨ Возможности

### 📋 Умные задачи
- Приоритеты (высокий / средний / низкий)
- Подзадачи и повторяющиеся задачи
- **Теги** для организации (работа, дом, важное)
- Свайп-жесты для быстрого завершения/удаления
- **Отмена удаления** через Snackbar
- Напоминания с точным временем
- Поиск по всем задачам

### 💰 Бюджет и финансы
- Учёт доходов и расходов по категориям
- **Круговая диаграмма** расходов и **столбчатый график** доходов/расходов
- **Цели накоплений** с прогресс-баром и пополнением
- **Регулярные платежи** (ежедневно / еженедельно / ежемесячно / ежегодно)
- Лимиты расходов по категориям
- Учёт долгов
- Оценка «здоровья» бюджета
- Финансовые снимки

### ⏱ Помодоро-таймер
- Настраиваемые интервалы фокусировки и перерывов
- **Фоновый сервис** — таймер работает при свёрнутом приложении
- Уведомление с оставшимся временем
- Статистика за день и за всё время

### 📝 Заметки
- Быстрый ввод текста
- Голосовой ввод
- **Поиск** по заголовку и содержанию
- **Отмена удаления**

### 🔒 Безопасность
- **PIN-код** при запуске приложения
- Настройка в разделе «Безопасность»

### 📱 Системные
- **Виджет** с балансом и задачами на сегодня
- Тёмная/светлая тема во всех экранах
- Экспорт данных в CSV
- Работа без интернета
- Минимальные разрешения

## 📥 Установка

### Скачать APK

👉 **[Скачать Todo Budget v4.0 (8.5 МБ)](https://github.com/emil-a-dev/todofin/releases/latest/download/TodoBudget-v4.0.apk)**

### Как установить

1. **Скачайте** APK по ссылке выше
2. **Разрешите установку** из неизвестных источников:
   - **Xiaomi/MIUI:** Настройки → Приложения → Специальный доступ → Установка неизвестных приложений
   - **Samsung:** Настройки → Приложения → Специальный доступ → Установка неизвестных приложений
   - **Другие:** Настройки → Безопасность → Неизвестные источники
3. **Откройте** скачанный файл и нажмите «Установить»
